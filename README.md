# 🖥️ System Health Monitor (Bash)

A simple **Bash-based system health monitoring tool** that detects your OS and lets you quickly check key system metrics like disk usage, memory usage, CPU load, and more — via an interactive menu.

---

## 🧠 Features

- Detects **Operating System** (Linux / macOS)
- Provides an interactive **menu**
- Check:
  - Disk usage
  - Memory usage
  - CPU load
- Uses real system tools: `df`, `free`, `top`, `uptime`
- Menu repeats until the user exits
- Can be extended with config and logging

---

## ⚙️ Requirements

This script is written in **Bash**, so it works on:

✔ Linux  
✔ macOS  
✔ WSL (Windows Subsystem for Linux)  
✔ Git Bash

❌ It does _not_ run natively in Windows.

---

## 🚀 How to Run

### Paste in terminal

- git clone https://github.com/print-aash/System_Health_Monitor.sh

- cd System_Health_Monitor.sh

- chmod +x System_Health_Monitor.sh

- ./System_Health_Monitor.sh
